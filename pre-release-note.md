# EffortOS 预发布说明

## 访问方式

### 用户体验实例
- 地址：http://localhost:9527
- 用途：直接体验完整功能
- 数据：使用 `effortos` 数据库（与测试隔离）

### 测试 API 实例
- 地址：http://localhost:19527
- 用途：API 调试和自动化测试
- 数据：同上（可随时清理）

### 启动方式
```bash
./dev_server.sh
```

## 已完成功能

### 核心功能
- **运动数据上传**：支持 TCX/GPX 文件上传，自动解析心率、功率、速度等数据
- **运动类型**：骑行、室内骑行、跑步、室内跑步、步行
- **Activity CRUD**：运动记录列表、详情查看、删除
- **运动指标计算**：TSS、NP、IF、VI、EF、做功（kJ）、分区时间统计
- **PMC 体系**：CTL（42天）/ ATL（7天）/ TSB 趋势追踪

### 用户参数
- FTP / 阈值心率（按运动类型） / 最大心率 / 体重
- 参数按日期生效，变更后自动标记受影响活动需重算
- 心率分区（Joe Friel 5 区）和功率分区（Coggan 7 区）自动推算

### 数据可视化
- **仪表盘**：今日 TSS、CTL/ATL/TSB 四宫格 + PMC 30 天趋势图 + 最近活动
- **活动列表**：上传、删除、分页
- **活动详情**：完整指标 + 心率/功率分区柱状图
- **参数设置**：参数配置 + 分区预览 + 历史记录

### AI 教练
- **训练周报**：基于运动科学理论分析（非直接交 LLM 自由发挥）
- **个性化建议**：支持自由提问
- 运动类型分布分析、负荷风险评估、趋势判断

## 建议体验重点

1. **上传运动文件** → 访问「活动」页面，上传 `samples/` 中的 TCX 文件
2. **设置参数** → 访问「设置」页面，配置 FTP 和 LTHR，观察指标变化
3. **查看仪表盘** → 首页 PMC 趋势图和 TSS 数据
4. **查看活动详情** → 点击活动查看分区图表和指标摘要
5. **AI 周报** → 访问「AI 教练」页面生成周报（需要配置 OPENAI_API_KEY）

## 已知限制

- 暂无用户认证系统，所有数据共享
- 样本数据需要自行上传（samples/ 文件夹中有示例文件）
- AI 功能需要 `.env` 中配置 `OPENAI_API_KEY`
