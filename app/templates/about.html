关于 EffortOS

科学训练，数据驱动，AI 赋能

项目介绍

EffortOS 是一款面向业余和半专业运动员的运动数据追踪与分析工具，类似 intervals.icu。 上传 TCX/GPX 运动数据文件，系统自动计算 TSS、CTL、ATL、TSB 等核心训练指标， 并通过 AI 教练提供个性化训练建议。

核心理念：用科学方法量化训练负荷，让你更聪明地训练，而不是更努力地训练。

核心功能

训练指标计算

TSS、CTL/ATL/TSB、NP、IF、EF、功率曲线、Best Efforts 等核心指标自动计算

数据可视化

PMC 趋势图、训练日历热力图、体能趋势、心率/功率分区图、活动对比

AI 教练

基于 OpenAI GPT 生成训练周报和个性化建议，回答你的训练问题

Wellness 追踪

日常主观感受 + HRV/静息心率记录，自动计算 Readiness 准备度分数

装备管理

追踪装备累计里程，设置使用寿命提醒，关联活动自动累计

数据导出

支持 CSV/JSON 格式导出活动数据，支持多维度筛选

技术栈
后端 Python + Flask
数据库 MongoDB
AI OpenAI GPT
前端 Chart.js + Vanilla JS
开源说明

EffortOS 同时是一个 AI 自驱动实验项目：以 Claude Code 为核心， 持续执行「规划 → 选取任务 → 实施 → 评估 → 再规划」的循环， 在只有明确目标而无详细设计的基础上不断自我进化。
